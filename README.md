# GRDBMacros
> Supercharge your `GRDB`'s `Record` with Swift macros!

## Requirements
- iOS 13.0+
- macOS 10.15+
- tvOS 13.0+
- watchOS 6.0+
- Swift 5.9+

## Installation
- In your `Package.swift`, add:
  ```
    .package(url: "https://github.com/ErrorErrorError/GRDBMacros.git", from: "1.0.0"),
  ```
